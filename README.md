# burger

The Burger app was created for the Node Express Handlebars homework.

## Deployment requirements

`npm i` will install the following dependencies:

* body-parser

* express

* express-handlebars

* method-override

* mysql

Then run the following in MySQL

* schema.sql

* seeds.sql
